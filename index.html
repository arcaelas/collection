ArcaelasCollection

Librería TypeScript para manipulación eficiente y tipada de colecciones de datos

ComenzarVer en GitHub
Instalación
npm install @arcaelas/collection

Potente API de Colecciones

Manipula colecciones de datos con una API fluida y expresiva. Filtra, ordena y transforma tus datos con métodos encadenables.

Tipado Fuerte

Desarrollado con TypeScript para ofrecer autocompletado inteligente y detección temprana de errores en tiempo de desarrollo.

Ligero y Rápido

Optimizado para rendimiento con cero dependencias externas. Ideal para aplicaciones que requieren alto rendimiento con colecciones de datos.
