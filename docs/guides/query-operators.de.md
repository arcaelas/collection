# Abfrageoperatoren

Meistern Sie MongoDB-Style-Abfrageoperatoren in `@arcaelas/collection`.

## Vergleichsoperatoren

### $eq (Gleich), $gt (Größer Als), $gte, $lt, $lte

## Logische Operatoren

### $not, $in, $contains

## Operator-Aliase

| Alias | Operator | Bedeutung |
|-------|----------|-----------|
| `=` | `$eq` | Gleich |
| `!=` | `$not` | Nicht gleich |
