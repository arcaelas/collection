# Operadores de Consulta

Domine los operadores de consulta estilo MongoDB en `@arcaelas/collection`.

## Operadores de Comparación

### $eq (Igual), $gt (Mayor Que), $gte, $lt, $lte

## Operadores Lógicos

### $not, $in, $contains

## Alias de Operadores

| Alias | Operador | Significado |
|-------|----------|-------------|
| `=` | `$eq` | Igual |
| `!=` | `$not` | No igual |
